This project was submitted by Alexis Contreras and Isaac Caro.
The project contains:

An include folder with all the necessary header files.
A src folder that contains the implementation of the header files.
There is an 
"images" folder which contains the images used to create the GUI.
The folder titled "txtfiles" contains the maze text file that contains the 20x20 maze.
The main.cpp file contains the main functions as well as the necessary 
functions required to build the adjacency list and prerform Dijkstras shortest path algorithm.
The Enemies.cpp and Enemies.h file runs the shortest path function 
The linkList.h and linkList.cpp files contain the necessary data
structures and functions used to contain the shortest path information.
The player, maze, and common src and header files contain relevant functions to initalize the elements of the maze.
